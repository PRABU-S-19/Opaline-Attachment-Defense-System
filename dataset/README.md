# Dataset Information

## Source
The training dataset used for the CANet (DistilBERT) model consists of
publicly available email corpora combining legitimate and malicious samples.

## Composition
- Total Samples : 10,000 emails
- Malicious     : 5,000 (phishing PDFs, macro DOCX, obfuscated scripts)
- Safe          : 5,000 (legitimate business documents)

## References
- CEAS 2008 Spam Dataset
- Enron Email Dataset (safe samples)
- PhishTank malicious sample repository

## Note
The actual dataset files are not included in this repository
due to size constraints. Download links are provided above.
